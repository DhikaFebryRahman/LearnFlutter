This project is a basic Flutter application created as part of a learning exercise to understand the fundamental concepts of Flutter development. The main purpose of this project is to introduce the structure of a Flutter project, including the use of widgets, layout organization, and basic application execution. The application demonstrates a simple user interface that runs on multiple platforms supported by Flutter. Through this project, learners can explore how Flutter handles UI rendering, project configuration, and file organization inside the lib directory. This project is intended for beginners who are starting their journey in mobile application development using Flutter and want to gain practical experience by building and running a simple application. The project also includes screenshots and documentation to help explain the development process and the final application result.
